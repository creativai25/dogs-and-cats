# Dogs & Cats — Petshop Premium Demo

Uma landing page moderna, responsiva e elegante projetada para um Petshop de alto padrão. O site apresenta serviços premium, cuidados especializados e um layout minimalista e atraente para tutores de cães e gatos.

## 🚀 Link de Acesso Online
O projeto está publicado e pode ser acessado em:
👉 **[https://dogs-and-cats-opal.vercel.app](https://dogs-and-cats-opal.vercel.app)**

---

## 🛠️ Tecnologias Utilizadas
- **HTML5** & **CSS3**: Estrutura semântica e estilização moderna.
- **Vanilla JavaScript**: Interações e efeitos dinâmicos na página.
- **Node.js**: Servidor estático leve local (`server.js`).
- **Vercel**: Plataforma de hospedagem e deploy contínuo.

---

## 💻 Como Executar Localmente

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado no seu computador.
2. Navegue até a pasta do projeto:
   ```bash
   cd C:\Users\mixba\.gemini\antigravity\scratch\petshop-demo
   ```
3. Inicie o servidor local:
   ```bash
   node server.js
   ```
4. Abra o navegador e acesse:
   👉 **[http://localhost:5180](http://localhost:5180)**

---

## 📦 Estrutura de Arquivos
- `index.html`: Página principal contendo as seções de serviços, depoimentos e formulários.
- `server.js`: Servidor local em Node.js nativo para desenvolvimento rápido.
- `vercel.json`: Arquivo de configuração de roteamento e deploy para a Vercel.
- `/frames`: Pasta contendo assets de mídia e imagens do site.
